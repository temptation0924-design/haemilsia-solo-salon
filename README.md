# 해밀시아 솔로 살롱 홈페이지

혼자 오는 사람만 받는 작은 술집 — Haemilsia Solo Salon 홈페이지.

## 기술 스택
- **프레임워크**: React 19 + TypeScript
- **스타일링**: Tailwind CSS 4 + shadcn/ui
- **빌드 도구**: Vite
- **라우팅**: Wouter
- **애니메이션**: Framer Motion

## 주요 기능
- 타로 색상표 기반 다크 퍼플/골드 테마 (Mystic Veil 빅토리안 고딕 살롱)
- 원페이지 스크롤 구조 (13개 섹션)
- 타로 카드 뽑기 미니게임 (14장 메이저 아르카나)
- 오픈 이벤트 카운트다운 타이머
- 마우스 반응 골드 파티클 (데스크톱)
- 모바일 반응형 + 햄버거 메뉴
- FAQ 아코디언
